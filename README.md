# tappedout-json
Chrome extension that allows you to export individual decks on http://tappedout.net/ as JSON

## Local Installation

`git clone https://github.com/nbrady-techempower/tappedout-json`

go to chrome://extensions/ and check off developer mode in the top right
then click on unpack extension and load the `tappedout-json` folder

Now when you go to a deck url like: http://tappedout.net/mtg-decks/26-05-15-athreos-on-the-river-styx/

You should see an `Export as JSON` button on the lower right corner:

![screenshot](https://i.imgur.com/j2qiCQi.png)
